About RPG Dungeon Game

RPG Dungeon Game is a simple text-based RPG game written in Python. The game is designed to provide a fun and interactive experience for players, while also serving as a learning tool for students interested in game development and learning Object Oriented Progrming.

Game Features

  • Player character with attributes such as health, armor, and attack power.
  • Enemies with varying levels of difficulty and attributes.
  • Combat system allowing players to attack or evade enemy attacks.
  • Leveling system allowing players to gain experience and upgrade their character.

Development

RPG Dungeon Game is currently being developed by students in Computer Science, Systems Analysis and Development, and other related fields. The game is open-source and available on GitHub.
